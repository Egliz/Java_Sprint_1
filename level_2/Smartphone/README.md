# Smartphone

**Descripción**:  
Ejercicio realizado para practicar la herencia y las interfaces, creando un objeto que hereda de una clase y que implementa varias funcionalidades mediante interfaces.

## 📌 Enunciado del ejercicio

Crear una clase 'Phone' con los atributos 'brand' y 'model', y un método para realizar llamadas.

Crear dos interfaces:

- Camera, con la función de hacer fotos.
- Clock, con la función de alarma.

Crear una clase 'Smartphone' que herede de 'Phone' e implemente las interfaces 'Camera' y 'Clock'.

Desde el main, crear un objeto 'Smartphone' y probar las tres funcionalidades: llamar, hacer una foto y activar la alarma.

## ✨ Funcionalidades

- Crear un teléfono con marca y modelo.
- Realizar una llamada.
- Hacer una foto.
- Activar una alarma.

## 🛠 Tecnologías

- Java
