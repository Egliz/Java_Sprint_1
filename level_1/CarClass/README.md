# Clase Coche




## Tecnologías
Java

##Investigación:
¿Cuál de estos atributos puede inicializarse desde el constructor?
El atributo 'power'(final) y para cada objeto existe un valor diferente.


¿Quién no puede modificarse nunca?
Los atributos 'power' (final) y 'brand' (static final), una vez se le asigna un valor ya no se puede cambiar en la ejecución.


¿Cuál afecta a todas las instancias de la clase?
Los atributos 'brand' (static final) y 'modelo' (static), todos los objetos creados tienen estos atributos en común.
