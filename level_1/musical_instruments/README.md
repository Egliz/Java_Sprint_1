**Descripción**: El objetivo de este proyecto es ver cómo podemos compartir características comunes entre diferentes objetos y, al mismo tiempo, hacer que cada uno se comporte de forma específica. 
Aprender cómo funciona la carga de clases en Java y los bloques estáticos.

## Investigación:
¿Qué es un bloque de inicialización (sin static)?


¿Cuándo se ejecuta?
Se ejecuta cada vez que se crea un objeto.

¿Qué diferencia hay entre éste y un bloque static?
El bloque static pertenece a la clase, solo se ejecuta una vez, el bloque de inicialización se ejecuta cada vez que se instancia un objeto, porque pertenece al objeto.
