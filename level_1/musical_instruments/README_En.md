# Musical Instruments
**Description**: The goal of this project is to explore how to share common characteristics among different objects while ensuring each one behaves in a specific way.
Learn how class loading and static blocks work in Java.

## ✨ Features:
Create different instruments.
Play each instrument (Emit sound based on the type).

## 🛠 Technologies:
Java

## 🚀 Installation and Execution:
1. Clone the repository: .
2. Open in IntelliJ or Eclipse.
3. Run Main.java.

## Research:
What is an initialization block (non-static)?

When does it execute?
It executes every time an object is created.

What is the difference between this and a static block?
The static block belongs to the class and executes only once; the initialization block executes every time an object is instantiated because it belongs to the object.
